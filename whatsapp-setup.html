إعداد WhatsApp

إعداد إرسال رسائل تأكيد الحجز عبر WhatsApp

1
إضافة رقم CallMeBot

أضف هذا الرقم إلى جهات الاتصال في WhatsApp:

+34 644 59 71 67
فتح في WhatsApp
2
إرسال رسالة التفعيل

أرسل هذه الرسالة بالضبط إلى الرقم:

I allow callmebot to send me messages
مهم: يجب كتابة الرسالة بالإنجليزية بالضبط كما هو مكتوب
3
الحصول على API Key

ستحصل على رد يحتوي على API Key مثل هذا:

CallMeBot: Your API key is: 123456
احتفظ بهذا الرقم - ستحتاجه في الخطوة التالية
4
تحديث إعدادات الموقع

افتح ملف firebase-config.js وحدث هذا الجزء:

const CALLMEBOT_CONFIG = {
    // ضع هنا رقم هاتفك (بدون + أو 00)
    phoneNumber: '966501234567', // مثال
    // ضع هنا API Key الذي حصلت عليه
    apiKey: '123456' // ضع الرقم الذي حصلت عليه
};
بدون + أو 00، ابدأ برمز البلد
الرقم الذي حصلت عليه من CallMeBot
